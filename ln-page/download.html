Download & setup

Clone the repo, load the extension, and connect your CLI tools.

  1. Clone

    git clone https://github.com/varshitha-sys/cloakwell-ai.git

  2. Load the browser extension

    Chrome → Extensions → Developer mode → Load unpacked → select the extension/ folder in this repo.

  3. Use it with Claude Code (or other CLI/agentic tools)

    CloakWell also intercepts CLI tools like Claude Code via a local TLS proxy, running the same detect/redact/block pipeline as the extension. Start it, then point the CLI tool at it in the same shell you'll launch it from: 

    ./setup.sh
source .cloakwell-env.sh
claude

    When you're done, stop the proxy so you don't leave a shell routed through it: ./setup.sh stop then unset HTTPS_PROXY, HTTP_PROXY and NODE_EXTRA_CA_CERTS.
